Обозначения (щелкните здесь, чтобы скрыть):
  Первая колонка содержит имя символа грамматики
  Вторая колонка содержит его тип:
 N  Нетерминал (его свойства далее маленькими буквами)
 s  – стартовый
 a  – аннулируемый
 l  – леворекурсивный
 r  – праворекурсивный
 c  – просто рекурсивный
 i  – косвенно (лево – после l, право – после r, просто – после c) рекурсивный
EOF  End of File
 T  Терминал (имя группы слов)
 W  Слово (конкретная строка)
 <action>  Действие
 <eps>  Ничего (пустая цепочка символов)
  Третья колонка содержит код символа (токен), используемый вместо этого символа в программе синтаксического акцептора/анализатора.
  Четвертая колонка содержит порядковый номер строки таблицы для установления соответствия строк со столбцами (не заполнена для действий и пустой цепочки).

Грамматика относится к классу LR(1)
Серым фоном отмечены клетки, в которых не удалось предупредить конфликты.
Черный цвет на сером фоне означает, что конфликт "сдвиг/свертка" условно разрешен в пользу операции сдвига.
Синий цвет на сером фоне означает, что конфликт "свертка/свертка" фактически не разрешен, хотя в клетке осталась только одна (случайно выбранная) операция свертки.
